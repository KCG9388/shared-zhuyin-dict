# 共享字庫

> 微軟注音「使用者造詞」的社群共享字庫。
> 讓大家一起維護字庫,打字更準、更順。

## 這是什麼

微軟新注音常常選錯字,內建詞庫缺很多現代詞(電動術語、專業詞…)。
「使用者造詞」可以補,但每個人各自造太累。這裡把大家整理好的字庫**集中分享**,
你只要下載套用,就能一次補進幾百個詞。

## 怎麼用(搭配「字庫管家」工具)

1. 用 **字庫管家** App 開啟「雲端模板」分頁
2. 挑想要的分類模板,或直接選「**全包**」(所有模板合併)
3. App 會先**自動驗證+修正注音**(補漏聲調、修槽位)+ 幫你備份目前字庫
4. 依指示在官方造詞工具按「匯入」→ 完成

> 不想挑?直接套 `integrated/all.txt`(全包)。
> 提醒:全包詞多而雜,若你只打特定領域,挑對應分類選字會更乾淨。

## 結構

```
index.json          模板清單(App 讀這個)
templates/          各分類字庫(UTF-16,詞+注音)
integrated/all.txt  全包:所有模板合併去重
```

字庫格式:每行 `詞彙　注音(每字含聲調)`,UTF-16 編碼。

## 想貢獻字庫?

歡迎!投稿方式(透過字庫管家 App 的上傳功能,審核後收錄)即將開放。
收錄原則:非內建、常用、讀音正確(工具會自動驗證)。
